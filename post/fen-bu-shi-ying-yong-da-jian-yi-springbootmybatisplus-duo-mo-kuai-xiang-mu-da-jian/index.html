分布式应用搭建（一）SpringBoot+MyBatisPlus多模块项目搭建

发布于 2020-03-29 | 分类于 SpringBoot+Dubbo分布式应用系列 | 15分钟 | 2704字数

前言

最近由于项目交接，开始进行新项目开发，框架选型为SpringBoot+Dubbo分布式应用。本片主要记录在IDEA中搭建SpringBoot多模块项目的过程。

开发工具及系统环境

  • IDE：IntelliJ IDEA 2019.3
  • 系统环境：mac OSX

项目目录结构

 . spring-boot-admin
│  pom.xml
│  README.md
│  
├─bootadmin-api（服务化工程对外暴露接口）
│    └─src.main.java
│         └─usys服务化节点名称
│             └─dto
│             └─service
├─bootadmin-core（服务化节点核心包）
└─bootadmin-usys（用户管理服务化节点）
      └─src.main.java
           └─usys包根路径
                └─entity
                └─mapper
                └─service.impl

搭建步骤

  1. 创建父工程
  • IDEA 工具栏选择菜单 File -> New -> Project...
  • 选择Spring Initializr，Initializr默认选择Default，点击Next
  • 填写输入框，点击Next
  • 这步不需要选择直接点Next
  • 点击Finish创建项目
  • 删除无用的.mvn目录、src目录、mvnw及mvnw.cmd文件，最终只留.gitignore和pom.xml
  1. 创建子模块
  • 选择项目根目录beta右键呼出菜单，选择New -> Module
  • 选择Maven，点击Next
  • 填写ArifactId，点击Next
  • 修改Module name增加横杠提升可读性，点击Finish
  • 同理添加其他子模块，最终得到项目目录结构如下图
  1. 搭建并运行

  • 在bootadmin-usys中添加com.ltlixer.bootadmin.usys.service.impl.UserServiceImpl.xml
package com.ltlixer.bootadmin.usys.service.impl;

import com.baomidou.mybatisplus.extension.service.impl.ServiceImpl;
import com.ltlixer.bootadmin.api.dto.UserDto;
import com.ltlixer.bootadmin.api.service.UserService;
import com.ltlixer.bootadmin.usys.entity.CoreUser;
import com.ltlixer.bootadmin.usys.mapper.CoreUserMapper;
import org.springframework.stereotype.Component;
import org.springframework.transaction.annotation.Transactional;

import java.util.ArrayList;
import java.util.List;

@Component
@Transactional
public class UserServiceImpl extends ServiceImpl<CoreUserMapper, CoreUser> implements UserService {

    @Override
    public List<UserDto> listUsers() {
        List<CoreUser> coreUsers = baseMapper.selectList(null);
        ArrayList<UserDto> userDtos = new ArrayList<>();
        UserDto userDto;
        for (CoreUser coreUser: coreUsers) {
            userDto = new UserDto();
            userDto.setUsername(coreUser.getId().toString());
            userDto.setMobile(coreUser.getAttachmentId());
            userDto.setSex("1");
            userDtos.add(userDto);
        }
        return userDtos;
    }

    @Override
    public void saveUser(UserDto user) {
        CoreUser coreUser = new CoreUser();
        coreUser.setName(user.getUsername());
        coreUser.setDelFlag(1);
        coreUser.setAttachmentId(user.getMobile());
        coreUser.setId((long) user.getUsername().hashCode());
        baseMapper.insert(coreUser);
    }
}

    }
}
